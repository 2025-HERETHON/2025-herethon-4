# 서비스 설명

## 기획 배경
<p align="center">
  <img width="1920" height="1080" alt="Image" src="https://github.com/user-attachments/assets/5f3640b6-9ad4-4272-b35f-ba52419b9f37" />
</p>

혼자만의 시간이 간절한데, 막상 어디로 가야 할지 막막했던 경험 있으신가요?<br/>
특히 여성들에게 '혼자만의 시간'은 때로 '불안감'과 함께 찾아옵니다.<br/>
<br/>

<img width="1920" height="1080" alt="Image" src="https://github.com/user-attachments/assets/088ace1a-edf2-45c3-ac31-df08bae2bdc7" />

실제로 여성 솔로 여행자의 73%가 안전을 걱정하고, <br/>한국 여성 중 '사회가 안전하다'고 느끼는 비율은 27.6%에 불과합니다.
<br/>

<img width="1920" height="1080" alt="Image" src="https://github.com/user-attachments/assets/e5f17672-b8e6-429b-bd5b-9a1431f88887" />

**혼자 있고 싶은데, 어디서 어떻게 안전하게 시간을 보낼 수 있을까?**

<br/>이 고민은 여전히 많은 여성들의 현실입니다. <br/>

**MOODYA** 는 바로 이 간극을 메우기 위해 시작되었습니다.<br/>

<br/>

**MOODYA는 단순한 장소 추천 앱이 아닙니다.** 

<br/>여러분의 **'감정'** 을 기반으로, 혼자만의 여정을 제안하고 온전한 **'회복'** 까지 도와주는 힐링 플랫폼입니다. <br/> 

**MOODYA** 가 제안하는 세 가지 탐험 여정을 소개합니다.

<br/>
<br/>

## 서비스 기능 상세 소개

<img width="1920" height="1080" alt="Image" src="https://github.com/user-attachments/assets/56dbefbe-a61a-4db9-93ec-d549103bf69f" />

**첫째**, 나를 위한 공간, 물리적 탐험입니다. <br/>

내 감정 상태와 활동 역량에 맞춰 서울 5개 권역 중 안전하고 편안한 장소를 추천해 드립니다.

<br/>
<br/>

<img width="1920" height="1080" alt="Image" src="https://github.com/user-attachments/assets/f05a62d9-2a4a-43dc-aa29-5d09de7bfe55" />

**둘째**, 내 마음을 들여다보는 감정 탐험입니다. <br/>

**"오늘 기분은 어때요?"** 간단한 감정 선택으로 여정을 시작하고, <br/>
활동 후에는 감정 회고를 기록하며 내 감정 흐름을 시각적으로 파악할 수 있습니다.

<br/><br/>

**셋째**, 나만의 회복을 기록하는 회복의 탐험입니다.<br/>

MOODYA의 리뷰 기능은 **오직 '나'를 위한 개인 기록 도구**입니다. <br/>
그날의 감정, 장소, 반응을 기록하며 자신을 이해하고 돌보는 나만의 루틴을 만들어갈 수 있습니다.

<br/><br/>

<img width="1920" height="1080" alt="Image" src="https://github.com/user-attachments/assets/1b04ce37-a555-4670-856e-33144fe3ae8d" />
기존 서비스들이 위치나 유행에 초점을 맞췄다면, <br/>

**MOODYA는 여러분의 '감정'에서 출발하여 '회복'으로 이어지는 진정한 여정을 설계합니다.**

<br/><br/>

감정 기반 맞춤 추천<br/>
감정 흐름 시각화<br/>
혼자만의 회복 기록<br/><br/>

MOODYA는  **'감정'** 이라는 언어로, 혼자 있는 시간을 불안함이 아닌 **안전하고 의미 있는 '나를 여행하는 시간'** 으로 만들어 드릴 것입니다.

<br/><br/>
